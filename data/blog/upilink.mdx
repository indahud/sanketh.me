[getupilink](https://getupilink.com) is tool to generate UPI payment link. It can be used to accept UPI payments with unique link.  

Users need to enter their merchant UPI ID and amount. Amount is optional.

Users can share their generated links or they can scan the QR code to proceed further.

It also offeres different tools:
- [Send whatsapp message without saving number](https://getupilink.com/tools/open-in-whatsapp)
- [UPI QR code generator](https://getupilink.com/tools/upi-qr-code-generator)  

### Link
[getupilink](https://getupilink.com)

### Tech Stack
- NextJS with Typescript
- Tailwind CSS
- React Hook Form
- Umami JS
